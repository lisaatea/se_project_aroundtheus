# Around the U.S.

This is the third project of the Software Engineering program at TripleTen. It was created using HTML, CSS, and Javascript, working from Figma design specifications for mobile and desktop layouts.

## Project features

- Grid Layout
- Flexbox
- Responsive Design
- Media Queries
- Responsive Modal
- Form Validation
